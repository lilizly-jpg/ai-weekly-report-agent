# 本周经营分析周报 ## 1. 核心结论 - 本周支付 GMV 为 12,486 万，环比下降 6.8%，主要由餐饮行业与北京城市贡献。 - 核销 GMV 环比下降 9.4%，跌幅高于支付 GMV，说明支付后到店履约存在压力。 - 商家动销率下降 3.6 个百分点，低动销商家池扩大，需进行商家分层运营。 - 退款率上升 2.4 个百分点，收入健康度需关注高退款商品和门店服务承接。 ## 2. 异动诊断 本周 GMV 下滑不是单一转化问题。支付转化率基本稳定，但内容侧直播 GMV 下滑、商家动销下降、核销率下降共同影响最终结果。 ## 3. 归因拆解 行业层面，餐饮行业下滑最明显；城市层面，北京贡献主要跌幅；内容层面，直播 GMV 下滑与开播场次减少有关；履约层面，核销率下降说明用户支付后到店链路存在断点。 ## 4. 策略建议 下周建议优先恢复头部商家直播供给，排查餐饮爆品套餐是否下架或价格带变化；同时针对未核销用户做触达提醒，并治理高退款商家与高退款商品。 ## 5. 下周关注 重点关注支付 GMV、核销率、商家动销率、退款率与直播 GMV。

面试讲解要点

你可以这样说
  • 这个页面不是单纯 Dashboard，而是一个经营分析 Agent 工作流。
  • 数字计算由规则层完成，AI 负责总结和表达，降低幻觉风险。
  • 业务链路按生活服务拆成商家、商品、内容、用户、支付、核销、售后。
  • 异动识别后按行业、城市、商家、商品、内容和履约进行归因。
  • 最终保留人工审核，适合经营周会和周报自动化场景。
