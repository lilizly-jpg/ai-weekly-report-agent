# AI Weekly Report Page

这是一个用于面试展示的“经营分析周报自动化 AI Agent”静态网页原型。

## 文件说明

- `index.html`：主页面，可直接通过浏览器打开。

## GitHub Pages 部署方式

1. 在 GitHub 新建一个 public repository，例如 `ai-weekly-report-agent`。
2. 上传本项目中的 `index.html` 和 `README.md` 到仓库根目录。
3. 进入仓库 `Settings` → `Pages`。
4. 在 `Build and deployment` 中选择：
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. 点击 `Save`。
6. 等待 1-2 分钟，GitHub 会生成一个可访问链接。

## 面试讲解

这个页面模拟生活服务经营分析周报自动化场景，将流程拆为：数据获取、指标计算、异动诊断、归因拆解、策略建议、AI 周报生成和人工审核。
